# Windows

categraf supports monitoring data collection not only on Linux but also on Windows, and the metric naming is identical, so alert rules and dashboards can be reused directly. No extra handling is needed for Windows.

## Installation

For installing categraf on Windows, refer to this [document](https://flashcat.cloud/docs/content/flashcat-monitor/categraf/2-installation/).

## Dashboards

Linux and Windows dashboards can actually be reused; only a few metrics differ between the two operating systems. For example, some metrics are Linux-specific, while others are Windows-specific.
